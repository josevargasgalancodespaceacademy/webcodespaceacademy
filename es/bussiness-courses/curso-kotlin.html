¿Que es Kotlin?

Kotlin es un lenguaje de programacion fuertemente tipado desarrollado por JetBrains. Ha sido fuertemente influenciado por lenguajes como Scala o C#. Además, targetea a la JVM (maquina virtual de Java 6) y a JavaScript de forma nativa. Es 100% interoperable con Java y permite proyectos mixtos mezclando código Java con código Kotlin. Además es compatible con Android, con iOS utilizando RoboVM y se puede targetear a Javascript. Por otro lado, es un lenguaje potente a la par que fácil de utilizar, la curva de aprendizaje es bastante más ligera que otros lenguajes y es un lenguaje ligero para poder ser ejecutado en cualquier dispositivo sin sobrepasar el peso específico de las APKs en Android. Por ello, es ideal para desarrollos basados en JVM y para hacer aplicaciones Android.

Descripcion y objetivos

El curso ofrece todo lo necesario para construir aplicaciones Android basadas en Kotlin, de forma rápida y sencilla.
