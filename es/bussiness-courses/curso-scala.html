¿Que es Scala?

Scala es un lenguaje de programacion multiparadigma, que une la programacion orientada a objetos y la programacion funcional, que promueven la escabilidad desde lo más pequeño. Además es un lenguaje orientado a objetos puro en el sentido de que cada valor es un objeto y cada operacion es una llamada a un metodo. El lenguaje soporta arquitecturas de componentes avanzados a través de clases y traits. Por otro lado, tambien es un lenguaje funcional en toda regla. Tiene todo lo que se puede esperar, incluyendo funciones de primera clase, funciones de orden superior, funciones aninadas, etc... Scala se ejecuta en todas las máquina virtuales de Java y tambien en Android, por ello las clases de Java y Scala pueden combinarse libremente. Por otro lado tiene una sintaxis concisa, elegante y flexible que minimiza la sintaxis innecesaria. Por todo esto, Scala brilla especialmente cuando se trata de software de servidor y facilita una escritura segura y una programacion multi-hilo de alto rendimiento.
